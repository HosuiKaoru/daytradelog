# 課題
1. 損切り合計額が増加傾向にある
2. 貸借銘柄の空売りの期待値がマイナスである
3. 資金管理が雑になっている
4. 根拠の乏しいトレードへ過剰投資している

<b><u>損切りへの対策</u></b>  
<span style="color: red;">ロットを下げる＋適切なin</span>  
ロットが大きいことが損切りしたくないという気持ちを無意識に助長している傾向あり。  
エントリーを遅らせたことで適切なイグジットができていない傾向あり。  
（寄り凸しないといけないのにひよって様子を見てから高い位置で入るなど）  
適切な位置でエントリーすることでプラン通りの出口戦略を実行できる。  

<b><u>空売りへの対策</u></b>  
<span style="color: red;">逆張りショート禁止</span>  
日足上＋分足上の状況で焼かれた。また、誤発注に気づかずに大きな損失を計上した。  
日足が上昇トレンド中は空売りを禁止とする。具体的には、日足が高値更新中 or 5MA上で推移中。  
誤発注はメイン画面に保有証券を配置することで対策する。  
あくまで下落トレンドを前提とした立ち回りとする。  

<b><u>資金管理への対策</u></b>  
<span style="color: red;">ロットの固定化を徹底する</span>  
これは合計ロット数であり、ナンピン等でこのロットを超える事態があってはならない。  
| 株価  | ロット上限 |
| -- | ------- | 
| 1000未満 | 1500 | 
| 3000未満 | 600 | 
| 5000未満 | 300 | 
| 10000未満 | 200 | 
| 10000以上 | 100 |  


<b><u>過剰投資への対策</u></b>  
<span style="color: red;">レバレッジ0.5倍を限度とする</span>  
新戦略の開拓は必須だがコストを減らすためにも上記のルールとする。  
株価が大きいほど勝率が低下傾向にあるので、値嵩株はそもそも言語化できていない戦略の適用を禁止することで対策する。  


# 月ごとの傾向
## 1月
連動スキャを徹底している。  
ボラの激しい日に損切りも大きくなるが結果として大きな利益が残せている。  
テーマ相場が日足上昇トレンド中に利益を出せている。  
連動性の否定の損切り遅れが大きめの被弾に繋がっている。  
また、低位株の大ロットJCとテーマ相場終了後の連動スキャで損失を計上している。  
損小利大ができている。  

## 2月
不正取引で大きな利益を獲得し、連動スキャルピングの日足の上昇トレンド中に利益を獲得している。  
しかし、あくまでスキャルピング気味にとっているわけで利幅はとれていない。  
寄り凸日和→上でJC死亡パターンの発現とテーマ相場死亡後の連動狙いで損失を計上。  
日足上＋分足上の状態での逆張りショートで大きく被弾している。  
上のデータだとRR改善されているように見えるが中身を見ると損大利小傾向にある。  
不正取引のおかげでRRよく見えるだけで、これを除くと傾向としては悪くなっていた。  
また、1月と比べてロットが大きくなっていることもわかった。  

## 3月
アルゴ銘柄のガラ拾いや値嵩株の謎トレードで利益を取る。  
だが結局、大型ではトータルマイナスを計上。普通にやられている。  
得意の逆張りも指値待ちをせずに、リバウンドした後に入って勝手に自滅している。  
順張りトレードを目指しているが大ロットでJC、仕込み玉は低ロットかつ利幅は少なめとかなりRRが悪くなっている。  
ロットは大きいまま推移。  

## 4月
日足の青天・反転戦略で利益を確保しているが、当然低ロットで利幅も最大とれているわけではない。  
3月よりは利伸ばしができているが、まだまだ利確に感情が出ている。  
一方で、寄り凸ひよりJCパターンの際はロットはフルレバという狂気。  
感情トレードの際も大ロットになっていて大きな被弾に繋がり、日足上＋分足上の逆張りショートも大きな被弾に繋がる。  
利益をとっているトレードはロットが小さく、大きな被弾の際はロットが大きいという最悪の状態。  
損大利小トレードの典型となっている。  

## なぜこのような傾向になったのか？
1. SNSでの収支報告への焦り
2. 負けパターンを認識しているもののそれを改善する具体策に欠けていたこと
3. FOMO

## どう改善するか？
1. SNSの収支報告は競争意識で見るのではなく、新たなトレード戦略を盗む視点で見る
2. 負けパターンの対応策を下記に記述したのでこれを徹底する
3. エントリー前に根拠を記入する

# 大きく被弾したトレード
## 4月
1. QPSホールディングス  
<b>逆張りショートで大きく被弾 → 感情的なドテン買いで追加損失計上</b>  
日足が高値更新中＋分足もGU上の状態でショートした。  
根拠は、アストロスケールが下落していたから連動して下落すると予想したため。  
逆行した時点で損切りしないといけなかったが反射的にナンピンしたことと、資金管理の甘さが招いた。  
逆張りショート禁止ルールによって対策。  

2. ブルーイノベーション/ヒーハイスト  
<b>寄り凸を日和って一段高でエントリー →  逆行した時点で損切りが遅れて連売りで被弾</b>  
関連銘柄の気配差を狙ったトレードは<span style="color: red;">寄り凸のみ</span>期待値があるためエントリータイミングがおかしい。  
また、そのまま吹き上げずに始値を割ってくる動きがあった時点でぶん投げないといけない。  
建玉が大きすぎたことが逃げる判断を遅らせ、この戦略への無意識レベルでの信頼度がないことが適切なinを躊躇させた。  
資金管理の徹底によって損切りの徹底を促し、月間トータルでの期待値を算出することでこの戦略への信頼獲得に繋げる。  

3. 日本ギア  
<b>スキャルピング狙いの空売りの建玉を忘れていて大きく被弾</b>  
保有証券をメイン画面に配置することで対策。  
   
4. リファインバース  
<b>日足が青天状態スタートの寄って一段高でJCで大きく被弾</b>  
大幅GU + 寄って一段高でそこから上げしろはほぼないのに大ロットで買い。  
すぐに逃げずに様子見して大きく被弾。  
新戦略でもないただの雰囲気トレード。  
こういう順張り雰囲気トレードへの過剰投資をロット数の上限を厳格化することで対策。  
また、各トレード根拠の事前エクセル記入をルーティン化することで雰囲気トレードかどうかを客観的に振り返るようにする。  

5. さくらインターネット  
<b>材料が出た当日に逆張りで被弾</b>  
貸借銘柄かつアルゴの振幅を狙った空売りをした。  
全く持って根拠に乏しいやり方。もし張り付かれたら危険すぎるトレード。  
材料当日の逆張りは特に危険なのでこれは完全に禁止にすることで対策。  

6. トライアイズ  
<b>日足の反転戦略を遅れてエントリー + ロット大で大きく被弾</b>  
当日大きく損失を計上していたため、取り戻したいという感情的な理由で分析が雑になった。  
朝一に被弾して6万円以上被弾したらロット上限を1/2にしてさらなる被弾を防ぐ。  

## 3月
1. 日本ギア/共栄タンカー  
<b>寄り凸を日和って一段高でエントリー →  逆行した時点で損切りが遅れて被弾</b>  
典型的な被弾パターン。3月にこの癖を見つけていたのに4月中に修正できなかった。  
5月以降は資金管理の徹底と試行回数を積むことだけを意識する。  

2. 古河電気工業/日東紡/フジクラ  
<b>先物の小陽線を見てロングするが下落トレンドは継続のためリバらずに被弾</b>  
先物連動狙いは無理だとここで徹底的に学んだ。  
指数が大幅GDスタートの時の銘柄選定も直近弱かった銘柄の方が寄り凸に期待値があるという発見ができた。  
4月はこれらのトレードをすることがほぼなかったので、5月以降は完全に0にする。  

3. 木村化工機  
<b>上昇トレンドのチャートが崩れたガラを拾いにいくが位置が高すぎて被弾</b>  
ガラに巻き込まれたので反射的に損切りする必要があったが反射的にナンピンしたことで傷口を広げた。  
ロットの大きさが判断の先延ばしに繋がった。  
上昇トレンドの継続を否定する形ではロングエントリーしないことを学んだ。  
4月はこのトレードは一度もしていないので今後も否定の形の時に流れに逆らうトレードをしないことを徹底する。
