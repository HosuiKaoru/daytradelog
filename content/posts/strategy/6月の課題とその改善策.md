# 課題
1. 入るべきじゃない場面で入って自滅するトレードが多くなっている
2. チャンスだと思っても入れていない or FOMO気味にエントリーが早い
3. 自分の考えが否定された時に損切りではなく反射的にナンピンをしてしまう癖がまだ残る
4. 利確を伸ばすのではなく損切りを伸ばす傾向にある
5. 最初に利益を作ってそこから削って終わる日が増えている

# 改善策
1. 根拠が明確に持てない時は入らないことを徹底する
2. チャンスだと思ったらちゃんと根拠を持った位置で指値でエントリーする（成り行き気味にエントリーすることがFOMOトレに繋がっているため）
3. 自分の当初の考えが否定された時は速攻で損切りする（ここに後付の理由禁止）
4. 予想通り動いてくれて含み益がうまれた時は一部を利確し、半分は引っ張るトレードを実施する（順張りの時は特に）
5. 最初に利益をしっかり作れた日こそ根拠を持てる時以外のエントリーを禁止する


# 新しく気づいた手法
1. 寄り天からの無慈悲な下落→後場にもう一段下げてきた時の当日の最安値狙い 
2. ソーサボトム上抜けロング
3. 引けにかけての酷い売りの時のクロオクショート狙い 
4. 大型株上昇トレンド株の日足の25MAでのピンポイントリバ狙い（スイング気味）
5. ロウソク足の形もその形成過程に注目するとよりテクニカルの信頼度が上がる

# 期待値のない手法
1. 5分足の当日高値更新後の2段上げチャートレ（脳死でこのパターンを使っても期待値がない。当該銘柄が当日の中で相対的な強さが感じられること、指数の一時的な下落に巻き込まれた形であることなどの他の複合的要因を組み合わせて発動させるべき）


# 7月どうする？
1. 大型中心の監視銘柄に変更する（新興に全くボラがないため）
2. とにかく当日強いものに入って順張りを引っ張るトレードスタイルに挑戦（Cis式）
3. 朝一のリバウンド狙いのチャンスを逃さないように前日準備を必ず毎日行う
4. 無計画なナンピンを一度もしないことを目標にする
5. 指値での注文をベースにする
6. 精鋭ログ・本・その他ブログからの情報収集を徹底する
